How it works

Three steps to explore your entire Snapchat history.

1

Request your data

Go to accounts.snapchat.com, navigate to My Data, and submit a download request. Make sure to include media.

2

Download & open

Download the zip file from Snapchat's email, then open Snap Data Explorer and select it.

3

Browse & search

Explore your chats, search messages, browse media, and view analytics. Everything stays local.

Troubleshooting

macOS: "App is damaged and can't be opened"

Apple blocks apps that aren't notarized (which costs $99/year). Since this is a free open-source project, you may see this error. To fix it, run this in Terminal:

xattr -cr /Applications/"Snap Data Explorer.app"

Build from source

Requires Node.js 20+ and Rust stable. See Tauri v2 prerequisites for platform-specific dependencies.

git clone https://github.com/KodyDennon/SnapDataExplorer.git
cd SnapDataExplorer
npm install && npm run tauri dev
